Roll-It PWA – Mobile-Ready Version

📱 How to Use:
1. Unzip and host the files on any HTTPS-enabled web server.
   - Example: GitHub Pages, Netlify, Vercel, Firebase Hosting

2. Open index.html in a modern mobile browser (Chrome, Safari, etc.)

3. To install on your phone:
   - Tap the browser menu (⋮ or Share icon)
   - Select “Add to Home Screen”
   - The app will install and run offline like a native app

🛠 Alternative Local Preview:
- You can also test it locally with Python:
  > python -m http.server 8080
  Then open: http://localhost:8080

Files included:
- index.html — Main app
- manifest.webmanifest — Metadata for install
- service-worker.js — Enables offline support
- icon-192.png — App icon for install

🎲 Enjoy the game!
